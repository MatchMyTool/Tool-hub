# MatchMyTool

A full-stack application with React frontend and Java backend.

## Project Structure

```
MatchMyTool/

├── backend/           - Java backend application
├── docs/              - Project documentation
└── README.md          - This file
```

## Setup Instructions

### Frontend (React)
Navigate to the `frontend/` folder and follow React setup instructions.

### Backend (Java)
Navigate to the `backend/` folder and follow Java backend setup instructions.

- Ensure both services are running for full-stack development
- Refer to individual README files in each folder for specific setup steps
